# E2E Placeholder Module

Dieses Modul (`:e2e`) ist als zukünftiger Ort für End-to-End-Tests vorgesehen.

Aktuell enthält es nur Build-/CI-Vorbereitung. Echte E2E-Tests werden erst implementiert, sobald die notwendigen Abhängigkeiten und technischen Grundlagen final verfügbar sind.

Zusätzlich enthält das Modul aktuell einen minimalen Dummy-Test, damit die Testausführung in CI für dieses Modul nicht leer ist.
